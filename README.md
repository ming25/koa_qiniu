# koa_qiniu
koa作为后台服务器，上传文件到七牛网站上



### 文件目录结构
koa_qiniu  
> bin  
>> www  
> public  
>> images  
>> javascripts  
>> stylesheets  
> routes  
>> index.js  
>> users.js  
> views  
>> error.ejs  
>> index.ejs  
>> upload_form.ejs  
>> uploader.ejs  
> README.md  
> app.js  
> config.js  
> package.json  



### 使用方法
```
> git clone git@github.com:ming25/koa_qiniu.git
> cd koa_qiniu
> npm install
> npm start
```
